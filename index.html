Code CBT

Your One Stop Code Questions Api

Code CBT, Whats That?

Well, simply put Code CBT is "coding questions at your fingertips".

Code CBT is a frontend api that provides you with coding questions for FREE which you can use as you will, for any purpose of your liking.

With Code CBT you can create code quiz/triva apps (checkout this demo), get questions to test your students (if you're a code tutor) and you dont have to bother making up questions to test yourself for an interview or interview potential employees.

What would I do with it

You can use code CBT for (but not limited to) the following purpose

  • Create amazing mobile, web or desktop code quiz/triva apps (checkout this demo)
  • Use the questions to test yourself on a personal basis
  • Test code parteners, collegues or students on their knowledge of a particular code topic
  • Prepare yourself for an interview
  • Answer questions for fun or practise
  • Practise or teach people how to make Http GET requests (Yeah!! Communicating with Code CBT's servers is good practise with making api calls ).
